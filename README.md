The Mission: Training an AI neural network to master Flappy Bird using the NEAT (NeuroEvolution of Augmenting Topologies) framework.

Core Tech: Powered by Python and Pygame, utilizing 2D bitmasks for pixel-perfect collision detection instead of standard bounding boxes.

The Brains: The AI receives real-time spatial data (bird position vs. pipe gaps) and evolves its network structure over generations to maximize its survival fitness.

Analytics: Includes integrated Matplotlib and Graphviz scripts to dynamically plot evolutionary stats and visualize the winning neural network topologies.
